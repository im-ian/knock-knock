# Knock Knock

**낙낙**은 Nextjs + Socket 테스트 용도로 제작된 프로젝트입니다.

데이터베이스 없이 소켓만 사용하여 접속 중이 아닌 경우의 메시지는 확인할 수 없어 장시간 접속을 장려(?)하는 프로젝트입니다.
